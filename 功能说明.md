# Blackout 功能说明

## 1. 软件定位

Blackout 是一个 Windows 托盘小工具，用来快速让屏幕变黑（用纯黑全屏窗口遮挡屏幕并把亮度降低到0），便于临时遮挡屏幕内容。

## 2. 基本使用方式

- 软件启动后常驻系统托盘。
- 用户通过托盘图标完成所有操作。

## 3. 需要具备的功能

### 3.1 黑屏模式

软件需要支持三种黑屏范围：

1. 全部屏幕黑屏
2. 仅主屏黑屏
3. 仅副屏黑屏

### 3.2 快速开关

- 左键点击托盘图标可以快速开/关黑屏。
- 黑屏状态下再次触发应恢复正常显示。

### 3.3 托盘菜单

右键菜单至少包含：

1. 一键黑屏
2. 主屏黑屏
3. 副屏黑屏
4. 重启软件
5. 退出软件

### 3.4 黑屏退出方式

黑屏时，至少支持以下退出方式：

- 按 Esc
- 鼠标点击黑屏区域
- 再次点击托盘图标

### 3.5 多显示器支持

- 能识别主屏和副屏。
- 在多显示器场景下，黑屏范围应与所选模式一致。
- 最好能识别所有显示器实现单独控制每一个屏幕的黑屏（能做到就做）

### 3.6 亮度体验

- 进入黑屏时，屏幕亮度应降低到0。
- 退出黑屏时，亮度恢复到黑屏前状态。

## 4. 结果要求

- 黑屏进入快、退出快。
- 黑屏和恢复过程稳定，不影响继续使用电脑。
- 重启与退出功能可正常使用。
